Masterwork Crafting Ledger
A specialized, interactive web tool designed for Neverwinter players to track and manage Masterwork crafting requirements. This application transforms complex material spreadsheets into a mobile-friendly, searchable interface.

📜 Overview
The Masterwork Crafting Ledger provides a centralized database of high-end crafting recipes for all character classes. Whether you are gearing up a Cleric, Paladin, or Warlock, this tool helps you identify exactly what you need and where to find it.

Key Features
Class-Specific Filtering: Quickly narrow down items relevant to your specific character class.

Real-Time Search: Search by item name or specific materials.

Interactive Material Guides: Click or tap any ingredient to reveal its specific drop location (e.g., Abyssal Hunts, Explorer Charts, or Master Trials).

Responsive Design: Optimized for use on a second monitor or mobile device while gaming.

🛠️ Built With
HTML5 / CSS3: Custom "Parchment & Ink" styling for an immersive RPG feel.

JavaScript (Vanilla): Light-weight, fast filtering and dynamic data rendering.

Google Fonts: Utilizing MedievalSharp and Lora for thematic typography.

🚀 How to Use
Search: Use the search bar to find a specific item like the "Duergar Mercenary's Steel Rapier."

Filter: Use the dropdown menu to see all available Masterwork items for your class.

Locate: Click on an ingredient row to see a popup indicating the exact location or activity required to obtain that resource.
